# Shortcuts

## Ctrl+Shift+P

    Command palette. Search any VS Code action by name.

## Ctrl+P

    Open any file by typing its name. Fast file switching.

## Ctrl+@

    Toggle built-in terminal open/closed.

## Ctrl+Shift+X

    Open the extensions marketplace to find and install extensions.

## Ctrl+Shift+E

    Open the explorer

## Alt+Up/Down

    Move the current line up or down.

## Ctrl+D

    Select the next occurence of the current word. Hit again to select more. Edit all at once.

## Ctrl+/

    Toggle comment on the current line or selection.

## F2

    Rename a variable everywhere it appears in the file at once.

## Ctrl+Shift+V

    Preview a markdown file rendered. Toggle between source and preview.

## Ctrl+B

    Toggle the sidebar. Hides the file explorer to give more coding space.
